# 匿名探索支持

一个为高中生提供匿名、非判断式性取向探索支持的网页工具。用户可以选择最接近自己当下困惑的问题，阅读安全、支持性的回应，并用匿名反馈帮助改进原型。

## 上线地址

https://my-idea-topaz.vercel.app/

## 本地打开方式

这个项目是一个静态网页，不需要安装依赖或运行构建命令。

1. 直接双击 `index.html`，用浏览器打开。
2. 或者在编辑器里用 Live Server 打开 `index.html`。

## 项目文件

- `index.html`: 网页主入口，也是 Vercel 部署的页面。
- `spec.md`: MVP 计划，说明项目目标、核心流程、反馈收集方式和成功标准。
- `research/`: 研究资料整理，包含研究问题、资料来源、摘要和数据表。

## MVP 功能

- 展示匿名和隐私说明。
- 让用户从四个常见困惑中选择一个。
- 显示对应的支持性回应和下一步建议。
- 收集最少量的匿名反馈：两个 1-5 分评分和一句开放反馈。
- 不收集姓名、学校、联系方式或精确位置。
