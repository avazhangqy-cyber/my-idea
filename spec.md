# 匿名探索支持 — MVP Spec

## 它做什么

一个匿名网页，让高中生选择一个关于性取向探索的困惑，并收到一段安全、非判断式回应。

## 给谁用

正在对性取向、隐私、安全表达感到困惑的高中生。

## 核心流程

1. 用户打开网页。
2. 用户选择一个最接近自己的困惑。
3. 页面显示对应的支持性回应。
4. 用户填写“是否被理解”“是否有帮助”和一个开放反馈。
5. 用户结束或重新选择。

## 不在 MVP 里

- 不做性取向测试
- 不判断用户身份
- 不收集姓名、学校、联系方式
- 不做登录系统
- 不做聊天机器人
- 不做社区功能
- 不做复杂数据分析

## 怎么收集反馈

收集用户选择的问题类型、两个 1-5 分评分，以及一个开放反馈问题。第一版可以先用页面内反馈或 Google Form，之后再接入 Google Sheets、Supabase 或 Firebase。
